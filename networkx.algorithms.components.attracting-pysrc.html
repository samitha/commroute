 21 -def attracting_components(G): 

 22      """Returns a list of attracting components in `G`. 
 23   
 24      An attracting component in a directed graph `G` is a strongly connected 
 25      component with the property that a random walker on the graph will never 
 26      leave the component, once it enters the component. 
 27   
 28      The nodes in attracting components can also be thought of as recurrent 
 29      nodes.  If a random walker enters the attractor containing the node, then 
 30      the node will be visited infinitely often. 
 31   
 32      Parameters 
 33      ---------- 
 34      G : DiGraph, MultiDiGraph 
 35          The graph to be analyzed. 
 36   
 37      Returns 
 38      ------- 
 39      attractors : list 
 40          The list of attracting components, sorted from largest attracting 
 41          component to smallest attracting component. 
 42   
 43      See Also 
 44      -------- 
 45      number_attracting_components 
 46      is_attracting_component  
 47      attracting_component_subgraphs 
 48   
 49      """ 
 50      cG = nx.condensation(G) 
 51      attractors = [scc for scc in cG if cG.out_degree(scc) == 0] 
 52      attractors.sort(key=len,reverse=True) 
 53      return attractors
