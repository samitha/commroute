 72      # The queue stores priority, node, cost to reach, and parent. 
 73      # Uses Python heapq to keep in priority order. 
 74      # Add each node's hash to the queue to prevent the underlying heap from 
 75      # attempting to compare the nodes themselves. The hash breaks ties in the 
 76      # priority and is guarenteed unique for all nodes in the graph.

 93              while node is not None:

 95                  node = explored[node]
